## 목적

원소 반응 시스템은 Project Aether의 전투 차별점입니다. 단순히 피해를 주는 공격이 아니라, 원소 상태를 쌓고 조합해 전투 결과를 바꾸는 구조를 보여주는 것이 목표입니다.

---

## 보여주려는 역량

| 항목 | 설명 |
|------|------|
| 상태 기반 전투 | 대상에게 남아 있는 원소 상태를 기준으로 다음 반응을 결정합니다. |
| 데이터 기반 규칙 | 원소 조합과 반응 결과를 코드에 고정하지 않고 데이터로 관리합니다. |
| 계산 흐름 분리 | 반응 판정, 피해 계산, 후속 이벤트를 서로 다른 단계로 나눕니다. |
| 피드백 연결 | 반응 결과가 화면 효과, HUD, 적 상태 변화로 이어지게 합니다. |

---

## v1 반응 범위

| 반응 | 목표 |
|------|------|
| 증폭형 반응 | 기존 공격 피해를 강화하는 흐름을 보여줍니다. |
| 상태형 반응 | 적의 행동이나 상태를 바꾸는 흐름을 보여줍니다. |
| 범위형 반응 | 여러 적에게 영향을 주는 전투 변화를 보여줍니다. |

---

## 완료 기준

- 원소 부착 상태를 확인할 수 있습니다.
- 새 원소 입력에 따라 반응이 결정됩니다.
- 반응 결과가 피해나 상태 변화로 이어집니다.
- 플레이어가 화면 피드백으로 반응 발생을 이해할 수 있습니다.
