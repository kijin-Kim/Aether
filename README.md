# Project Aether

원소 반응 기반 오픈월드 액션 RPG. 원신의 핵심 재미 요소를 언리얼 엔진으로 재해석한다.

---

## 핵심 기술

1. 오픈월드 (World Partition)
2. 원소 전투 시스템 (GAS)
3. AI (StateTree·BT·SmartObject)
4. 퀘스트 시스템
5. UI 시스템 (최적화)


## 핵심 재미 루프

```
탐험 발견 → 원소 전투(GAS 콤보) → 성장(캐릭터·장비) → 더 깊은 탐험
```
